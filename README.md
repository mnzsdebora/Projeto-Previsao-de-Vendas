# Projeto Previsão de Vendas
Esse projeto tem como objetivo analisar a receita de uma empresa ao longo do tempo e realizar previsões de receita.


---

## **Descrição do Projeto**

A predição é feita a partir de variáveis como:
- Receita
- Sazonalidade
- Período
- Entre outros.

Foi realizada uma análise exploratória e o pré-processamento dos dados para garantir a qualidade das informações usadas nos modelos.

---

##  **Técnicas e Algoritmos Utilizados**

### **Modelo de Machine Learning**
-  Random Forest Regressor

### **Pré-processamento**
- Limpeza de dados
- Tratamento de valores ausentes
- Sazonalidade
